# 🛡️ Cyber Attack Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard that analyzes cyber attack data to identify trends, attack types, affected industries, financial losses, and country-wise distribution. The dashboard provides clear visual insights to support data-driven decision-making.

## Tools Used

- Power BI
- SQL
- Microsoft Excel

## Project Workflow

- Data Collection
- Data Cleaning
- Data Analysis
- Dashboard Development
- Data Visualization

## Dashboard Features

- Cyber Attack Overview
- Attack Type Analysis
- Industry-wise Analysis
- Country-wise Analysis
- Financial Loss Analysis
- Interactive Filters

## Key Insights

- Identified the most common cyber attack types.
- Compared attacks across industries and countries.
- Analyzed financial losses caused by cyber attacks.
- Visualized trends using interactive charts and KPIs.
